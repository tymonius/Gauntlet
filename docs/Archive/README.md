# Documentation Archive

This directory contains completed audits, superseded working notes, and historical development records that remain useful for provenance but are not active sources of truth.

Files in this directory:

- may explain how a current decision was reached;
- may preserve prior terminology, intermediate proposals, or artifact logs;
- must not override current rules, canonical release data, active card-review decisions, or the Project Index;
- should be cited as history rather than treated as current instructions.

Current development should begin with `docs/Gauntlet_v0.6_Project_Index.md`.
