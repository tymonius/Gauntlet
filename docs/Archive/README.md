# Documentation Archive

This directory contains completed audits, superseded working notes, migration records, and historical development directions. Archive material preserves provenance but is never an active source of truth.

Current work should begin with:

- [Documentation Index](../README.md)
- [Development Status](../Gauntlet_Development_Status.md)
- [v0.6.0 Release Package](../../releases/v0.6/README.md)

## Archive groups

### `v0.6-development/`

Pre-release v0.6 working rules, the preliminary core rulebook, the former Project Index and Open Questions trackers, consolidated card metadata, migration logs, completed card reviews, and all Territory review records.

These files intentionally preserve retired terminology and intermediate mechanics. The official v0.6.0 rulebook and definitive card sources supersede them completely.

### `v0.5.7-development/`

Clarifications and development records from the final pre-faction rules generation. The versioned v0.5.7 release remains under `releases/v0.5.7/`.

### Other archived records

Conversation audits, character-design logs, superseded visual notes, and long-form development history remain at the archive root when they span several versions or do not belong to one release cycle.

## Archive rules

Archived files:

- may explain how a current decision was reached;
- may contain old terminology, unresolved proposals, or obsolete exact text;
- must not override current rules, canonical data, definitive faction guides, or active source-card documents;
- should not be edited merely to match current terminology;
- should be cited as history rather than instructions.
