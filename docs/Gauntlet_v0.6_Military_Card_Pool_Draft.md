# Gauntlet v0.6 Military Card Pool — Superseded First Pass

**Status:** Historical draft; not an active source of Military costs, text, roster selection, or package recommendations.

The first complete twelve-card concept draft that previously occupied this file has been superseded by the completed Military documentation:

- `Gauntlet_v0.6_Military_Card_Pool.md` — authoritative current roster, costs, complexity, uniqueness, exact text, package audit, and playtest watchlist;
- `Gauntlet_v0.6_Military_Design_Notes.md` — active identity, strategic rationale, leader integration, weaknesses, and package-level testing priorities;

Most of those designs were rejected or substantially rebuilt during card-by-card review.

The full original text remains available through this file's Git history. It does not override the active Military Card Pool.
