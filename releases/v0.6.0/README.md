# Gauntlet v0.6.0 Faction Framework Release

Canonical pre-release package generated July 20, 2026.

Gauntlet v0.6.0 introduces six complete factions, twelve Leaders, 72 faction cards, and the first unified rulebook for the faction era. This package supersedes earlier releases for current playtesting. Earlier release files remain available as historical records.

The Gauntlet project repository, current release materials, and hosted browser tools are available at **[gauntlet.run](https://gauntlet.run/)**.

## Factions

1. Military — General or Commandant
2. Diplomats — Ambassador or Senator
3. Financiers — Banker or Executive
4. Intelligence — Ranger or Spymaster
5. Mystics — Alchemist or Spirit Walker
6. Inquisition — Grand Inquisitor or Witch Hunter

## Printing

Printable card sheets use standard 2.5 × 3.5 inch poker-card fronts in 3 × 3 US Letter layouts. Print at **Actual Size / 100%** with scaling disabled. Sleeve cut fronts in opaque poker-size sleeves with ordinary playing cards or bulk trading cards behind them.

Territory cards use the same physical card size and are oriented so that their facing indicates control.

## Source hierarchy

1. `Gauntlet_v0.6.0_Rulebook.md` governs shared rules.
2. Definitive faction guides govern faction-specific rules and exact faction-card text.
3. The Neutral and Territory pool documents govern their exact player-facing text.
4. Generated canonical data mirrors those sources and must not be edited independently.
